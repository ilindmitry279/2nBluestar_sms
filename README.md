# 2nBluestar_sms
Sms sender for 2n Bluestar GSM gateway.

usage:

sms in push format:

sms.py XXXXXXXXXX "Text message" y

XXXXXXXXXX - tel. number оf the recipient

y - sms in push format

sms in standart format(not push):

sms.py XXXXXXXXXX "Text message"

XXXXXXXXXX - tel. number оf the recipient
